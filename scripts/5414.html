Saturated arithmetic 2.0

2014-09-08 by Public Domain

download Saturated arithmetic 2.0.ps1

Saturated addition, subtraction, multiplication, and casting from numeric types to integer types using clamping to handle overflows.

<#

Version 2.0:
- fixes a couple of integer widening bugs
- adds subtraction and multiplication
- changes syntax to (where [T] is a primitive integer type):

[sat[T]]::From($a)
[sat[T]]::Add($a,$b)
[sat[T]]::Sub($a,$b)
[sat[T]]::Mul($a,$b)

Instead of the older syntax:

[sat]::iWhatever($a)
[sat]::uWhatever($a)
[sat]::add_Whatever($a,$b)

#>
