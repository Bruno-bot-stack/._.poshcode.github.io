QAD.psm1

2011-05-24 by Joel Bennett

download QAD.psm1.ps1

This is the file you need to load Quest's Active Directory snap-in as a module. Just put this in a folder with all the dlls ...

I'm posting it as an example of what my "Reflection":http://poshcode.org/2480 module's New-ModuleManifestFromSnapin function can do.
